セッション終了時、worklog.md追記とファイルコピーだけで終わらせてはいけない。

**Why:** 別のローカル環境でもスムーズに作業を引き継げるようにするため。ローカルmemoryファイルの内容が古いままコピーしても意味がない。2026-04-16のセッションでPhase 1-4の設計判断7項目をメモリに書き込み忘れて指摘を受けた。

**How to apply:** セッション中断/終了時のチェックリスト:
1. そのセッションで確定した設計判断・新知見をローカルmemoryファイルに追記
2. ローカルmemoryから .claude/memory/ へコピー
3. worklog.md に作業記録を追記
4. git add → commit → push
5. 「同期済みか」を自問してから終了報告する
