全日本かるた協会の公認大会で**申込が定員超過した場合、主催者枠(20%以内)を除く残枠を、その年度の出場回数が「少ない」選手から優先**割当し、同回数は公開抽選で決める制度(2024年4月/令和6年度〜・当面A級のみ)。「出場回数による優遇」だが**多い人有利の逆＝少ない人優先(機会分散)**。

**出場回数の定義(正式通達ベース・最精密は「予定表について＿補足説明」2024-09-09)**:
- 期間=**年度4月〜翌3月**(年度替りでゼロリセット)。⚠️A級得点の年度10/1〜9/30とは別物。
- 対象大会=**今年度の公認大会＋新春大会のみ**(新春以外の主催大会・後援大会は対象外)。
- 基準=申込先大会の**申込開始日(≒案内発信日)の前日まで**の「**出場選手確定者名簿掲載**＋抽選大会の**当日繰上り出場者**」の合計。確定名簿掲載は**後でキャンセルしてもカウント**/繰上り出場カウント/**繰上り辞退は対象外**/落選はカウントされない。
- B→A昇級者は今年度B級出場回数(公認・新春)も加算。自己申告(申込書記入,A/B/C級)＋協会が抽選結果+結果報告で集計しクロスチェック。

**出典**: 公開=競技会規程第二条三(令和6年4月1日, governanceページ)。詳細定義は**会員ページ限定の通達**=「公認大会『抽選制度』についてのお知らせ」`member-document/13161`＋「公認大会新制度の概要」`member-document/10819`の各添付(docx/xlsx/pdf)。会員ログインは [[reference_karuta_member_page_credentials]]、積極利用は [[feedback_karuta_member_page_proactive]]。
- 別制度との混同注意: 名人・クイーン東西予選は出場"回数"でなく**段位→A級得点(成績)→抽選**。
